# Task 8 – Capture Network Traffic with Wireshark #
Objective

Capture and analyze network traffic using Wireshark, focusing on HTTP packet inspection.

Tools Used

Kali Linux

Wireshark

Web browser (for generating HTTP traffic)

# * Steps Performed * #
# 1. Launched Wireshark #

Selected the active network interface and started packet capture.

# 2. Generated HTTP Traffic #

Visited:

http://example.com OR http://your_metasploit_id


to create HTTP requests and responses.

# 3. Applied HTTP Filter #

Used the filter:

http

to extract only HTTP packets from the capture.

# 4. Analyzed Packet Details #

Inspected:

HTTP GET requests

HTTP responses

Host and User-Agent headers

Status codes

Server information

# 5. Saved the Packet Capture #

Saved as:

wireshark_capture.pcap

Included Files

wireshark_capture.pcap – Packet capture file

/screenshots – Optional screenshots of filtering and packet inspection

VIDEO_LINK.txt – Link to demonstration video

# Conclusion #

Wireshark is a powerful tool for capturing and analyzing network traffic.
This task demonstrates how HTTP communication works at packet level—valuable for network security analysis.
